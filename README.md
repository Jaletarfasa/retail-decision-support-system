# Retail Decision Support System — Full-Version Starter

This is a modular starter architecture for evolving the local retail decision support prototype into a fuller version.

It keeps the honest maturity framing:
- local prototype foundation
- modularized pipeline
- retrieval and routing remain local/source-aware
- no fake enterprise claims

Run order:
1. configure YAML files in `config/`
2. populate / customize source modules in `src/`
3. run the orchestrator script
4. inspect artifacts in `artifacts/runs/`
5. launch the Streamlit app
